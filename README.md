
# TorrServer Global Finder PRO

Сайт для поиска рабочих TorrServer серверов.

Работает полностью через GitHub Pages.

## Возможности

✔ генерация новых IP при каждом поиске  
✔ проверка ~2000 серверов за запуск  
✔ показывает до 50 рабочих серверов  
✔ отображает ping  
✔ показывает страну и город  
✔ кнопка копирования  
✔ удобный интерфейс для телефона  

Формат:

IP:PORT | ping ms (country / city)

пример:

91.193.43.141:8090 | 110 ms (Germany / Berlin)

## Запуск через GitHub

1. Создайте репозиторий на GitHub

например

torrserver-global-finder-pro

2. Загрузите файл

index.html

3. Откройте:

Settings → Pages

4. Включите:

Deploy from branch → main

5. Ваш сайт будет:

https://USERNAME.github.io/REPOSITORY

## Использование

1. открыть сайт  
2. нажать кнопку "Обновить серверы"  
3. сайт сгенерирует новые IP  
4. проверит ~2000 серверов  
5. покажет рабочие  

Адреса можно вставлять в Lampa.
